    # 快速开始 ✨

    # Fork本项目到您的账号或组织下

    这是一个简单的过程。分叉版本库的步骤如下。

    • 登录到GitHub账户。
    • 打开本GitHub仓库。
    • 点击版本库页面右上方的Fork按钮。

    注意

    https://gitee.com/luch4736/Phigrim-Docs´是本仓库的中国大陆镜像，请勿在gitee中Fork本项目。

    # 拉取此项目到本地 💻

    git clone https://github.com/user/Phigrim-Docs.git

    然后，开始编辑您的内容，并提交Pull Request。

    # 简介 🔥

    这是一个基于VuePress的文档项目，由luch4736创建，旨在通过社区创造规范。

    您可以修改/src目录下的.md格式的文档，或者在issue中提出您想要加入的功能。

    # 提交您的编辑 🙋

    Phigrim Docs在Github使用MPL2.0协议开源。您可以在这里open in new window请求合并您的编辑。

    # 评论 🔨

    在Phigrim Docs中，除了主页的所有文档都开启了评论系统。

    你可以对当前文档发表评论，或在Github中的本项目链接中找到Discussions修改你的评论。

    目前，main分支的管理并没有完全统一，所以你可能看到混乱的推送信息
